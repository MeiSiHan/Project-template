## 创建koa2-generator
### 1 全局安装koa-generator
``` javascript
npm install -g koa-generator
```
### 2 创建项目
项目创建命名 koa2 -e 项目名
```
koa2 -e template-koa2-node
```
### 3 进入项目和安装依赖
```
cd template-koa2-node && npm install
```
### 4 安装 mongoose
```
 npm install mongoose --save-dev
```
### 5 运行项目
```
 npm run start
```
### 6 命令运行
```
 SET DEBUG=koa* & npm start template-koa2-node
```
### 7 打开浏览器
`localhost:3000`
